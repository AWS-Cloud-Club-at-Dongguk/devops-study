Markdown으로 문서를 꾸밀 때 사용할 수 있는 몇 가지 예쁜 요소들:

### 1. **헤더**
```markdown
# 큰 제목
## 중간 제목
### 작은 제목
```

### 2. **강조**
```markdown
*기울임* 또는 _기울임_
**굵게** 또는 __굵게__
~~취소선~~
```

### 3. **리스트**
- 순서 없는 리스트:
    ```markdown
    - 항목 1
    - 항목 2
        - 하위 항목
    ```
- 순서 있는 리스트:
    ```markdown
    1. 첫 번째
    2. 두 번째
    ```

### 4. **링크와 이미지**
```markdown
[링크 텍스트](https://example.com)
![이미지 설명](이미지_URL)
```

### 5. **코드 블록**
```markdown
`인라인 코드`
```
````
```언어
코드 내용
```
````

### 6. **표**
```markdown
| 헤더1 | 헤더2 | 헤더3 |
|-------|-------|-------|
| 데이터1 | 데이터2 | 데이터3 |
```

### 7. **인용문**
```markdown
> 인용문
>> 중첩된 인용문
```

### 8. **수평선**
```markdown
---
***
```

### 9. **체크리스트**
```markdown
- [ ] 할 일 1
- [x] 완료된 일
```

### 10. **이모지**
```markdown
🎉 ✨ 👍
```

이 요소들을 조합하여 문서를 예쁘게 꾸밀 수 있습니다!
